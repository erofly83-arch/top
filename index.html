Как загрузить и обработать файлы
1
Есть файл-память настроек с прошлого раза? Нажмите карточку «Файл памяти» — настройки колонок и база кросскодов загрузятся автоматически
2

Мониторинг цен поставщиков
Сводная таблица, где ваш прайс и прайсы поставщиков встречаются в одном месте. Сравнивайте цены, выявляйте разницу и принимайте решения — без ручного сведения в Excel.
Сравнение по штрихкоду
Товары из разных файлов сопоставляются по штрихкоду. Если коды отличаются — используется база кросскодов из файла-памяти настроек.
Несколько поставщиков
Каждый загруженный файл поставщика — отдельный столбец с ценой. Сразу видно, у кого выгоднее по каждой позиции.
Мониторинг по типу оплаты
У поставщика есть нал и безнал? Выберите обе колонки при загрузке — они отобразятся как отдельные столбцы и сравниваются раздельно.
Дельта % — точный фильтр цен
Кнопка «Δ Дельта %» позволяет выбрать две колонки для сравнения и задать диапазон отклонения. Например: показать только товары, где лучший поставщик дешевле вашего прайса на 5–30%. Выбранные колонки подсвечиваются серым в таблице.
Фильтры и экспорт
Фильтруйте по разнице цен, поиску новинок или компактному виду. Экспортируйте результат в Excel одной кнопкой.
Как начать
1
Перейдите на вкладку «Загрузка прайсов» и загрузите ваш прайс и файлы поставщиков
2
Для каждого файла выберите нужные колонки и нажмите «Готово — передать в мониторинг»
3
Вернитесь сюда — таблица заполнится автоматически
Если у вас уже есть файл-память с прошлого раза — загрузите его первым на вкладке «Загрузка прайсов». Он восстановит разметку колонок и базу кросскодов.

Поиск кросскодов Beta
Инструмент для связывания товаров с разными штрихкодами. Если один и тот же товар у вас и у поставщика обозначен разными кодами — здесь вы найдёте и зафиксируете эту связь.
Умный матчинг
Система сравнивает наименования товаров между файлами и предлагает пары с оценкой схожести от 0 до 100%.
Ручное подтверждение
Вы сами решаете, какие пары верны. Подтверждённые связи сохраняются в базу и сразу учитываются в мониторинге.
Накапливаемая база
Связи хранятся в файле-памяти настроек. При следующей загрузке все найденные кросскоды применяются автоматически.
Как использовать
1
Убедитесь, что прайсы загружены и переданы в мониторинг на вкладке «Загрузка прайсов»
2
Нажмите «Запустить поиск» — система проанализирует наименования и составит список кандидатов на совпадение
3
Просмотрите предложенные пары. Фильтруйте по уровню уверенности: ≥80% — высокие, 55–79% — средние, 35–54% — возможные (отдельная вкладка)
4
Нажимайте «✓» на верных парах — они попадут в базу и сразу улучшат сводку в мониторинге
5
Скачайте обновлённый файл-память через «Скачать всё архивом», чтобы сохранить базу для следующей сессии
Используйте этот инструмент после первичного мониторинга — когда видите строки без совпадений и хотите их связать.

База кросскодов — штрихкоды
Здесь хранятся группы эквивалентных штрихкодов. Если один и тот же товар у разных поставщиков имеет разные коды — добавьте их в одну группу, и при сравнении они будут считаться одним товаром.
Группа кросскодов
Каждая запись — это товар с главным штрихкодом и списком альтернативных кодов поставщиков. При мониторинге все коды из группы считаются одним товаром.
Загрузка из файла-памяти
Если вы уже работали с приложением раньше и скачали файл-память — загрузите его на вкладке «Загрузка прайсов» и база восстановится автоматически.
Пополнение через поиск
Используйте вкладку «Поиск кросскодов» — система предложит пары товаров по схожести наименований. Подтверждённые пары попадут сюда автоматически.
Как добавить запись вручную
1
Заполните строку выше: название товара, главный штрихкод (ваш основной код), кросскоды — коды того же товара у поставщиков, через запятую
2
Нажмите «Добавить» — запись появится в таблице
3
После наполнения базы нажмите «Скачать файл-память», чтобы сохранить всё для следующей сессии
# Название Главный ШК Кросскоды (Enter для добавления)
База брендов и смысловых групп
Словарь помогает сопоставлять товары, которые в разных прайсах называются по-разному. Используйте его не только для брендов — заводите группы для вкусов, цветов, типов упаковки и любых других признаков: «чёрный», «зелёный», «пэт», «пакет», «карамель» и т. д.
Канонический ключ
Главное название группы — как вы хотите видеть её в базе. Система будет искать товары, содержащие этот ключ или любой из его синонимов.
Синонимы
Альтернативные написания того же понятия. Если в прайсе поставщика встречается любое из этих слов — товар относится к данной группе. Указывайте все варианты: на русском, латинице, с опечатками.
Антонимы
Слова, которые исключают принадлежность к группе. Если в наименовании есть антоним — товар не попадёт в эту группу, даже когда синоним тоже присутствует. Используйте антонимы, чтобы разграничить похожие группы.
Конфликты
Одно слово не может быть синонимом сразу в двух разных группах — это противоречие. Система помечает такие пересечения жёлтым значком ⚠️, чтобы вы могли их исправить.
Как добавить группу
1
Нажмите «Добавить бренд» и введите канонический ключ — однословное название группы: «чёрный», «nescafe», «пэт», «пакет»
2
В поле «Кросскоды» через запятую укажите все альтернативные написания этого слова — кириллица, латиница, возможные опечатки
3
В поле «Антонимы» укажите слова, которые исключают эту группу. Для «чёрного» — это «зелёный», «белый», «красный» и их англоязычные варианты
4
Нажмите «Сохранить» и скачайте файл-память — при следующей загрузке все группы применятся автоматически
